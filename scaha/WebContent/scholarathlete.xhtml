Dear SCAHA Membership,



The 2015 SCAHA Scholar Athlete program is open for registration. To qualify for the Scholar Athlete program the player must have a 3.0 or equivalent GPA or a minimum of "S" (satisfactory) in citizenship and effort.



To register complete the following steps:



1. Login to your account on www.SCAHA.com

2. Select the Scholar Athlete link.

3. Select the Upload Report Card button.

4. Select File and select Upload button to upload the selected report card to the site.

5. Once report card is uploaded select Submit Scholar Athlete Application button.





The applications will be reviewed by SCAHA staff, once approved the players name will appear on the Scholar Athlete page of the site and the player will receive a Scholar Athlete patch in the mail.



All scholar athlete recognition patches are awarded at the sole discretion of the chairperson in conjunction with the SCAHA Executive Committee. There will be no appeals and all decisions on awards are final.
